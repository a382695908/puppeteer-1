Muppet Entity

This is the Muppet Entity. It is responsible for the managment of Muppets (in the form of a Collection or a Model). They are stored under the Entities module.

@App.module "Entities", (Entities, App, Backbone, Marionette, $, _) ->

Muppet Model

Represents a single Muppet. Each of them should have a name an image and a description, but we don't set them explicitly as defaults since our data is being bootstraped for this sample application.

class Entities.Muppet extends Entities.Model

Muppets Collection

Represents a group of Muppets.

class Entities.Muppets extends Entities.Collection model: Entities.Muppet

Filter by name

filterByName: (text) ->

If no text is provided, returns all Models in the Collection.

return @models if text is ""

Else, returns only models which contains text on its name attribute (case insensitive).

return @filter (muppet) -> muppetName = muppet.get("name").toUpperCase() return _.include muppetName, text.toUpperCase()
